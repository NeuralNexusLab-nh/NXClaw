NXClaw
Autonomous Sandboxed Terminal Agent

System Overview

NXClaw is a minimalist, zero-dependency command-line AI agent operating autonomously inside a secure, sandboxed workspace. Built using native Python standard libraries, it executes commands, edits files, and safely patches scripts without complex third-party framework overhead.

Deployment & Execution

Windows Setup

Download and move the compiled binary executable to your targeted workspace folder. Launch it through standard CMD or PowerShell terminals:

> .\nxclaw.exe
Linux Setup

Download the executable wrapper script. Grant executable permissions via standard shell modes and run to boot up the backend parser:

$ chmod +x nxclaw.sh
$ ./nxclaw.sh
Python / macOS Setup

Verify that Python 3.8+ is installed on your local host system. Download the source file and boot up immediately without pip installations:

$ python3 nxclaw.py
